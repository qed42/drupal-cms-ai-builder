# TASK-277: Onboarding UX Polish — Skip Labels, Helper Text, Layout Variation

**Story:** ONB-3, ONB-4, ONB-5
**Priority:** P1
**Effort:** M
**Sprint:** 17

## Description

Polish the onboarding wizard with trust-building micro-interactions.

### Deliverables

1. **Skip/Optional labels (ONB-3):** Steps that accept optional input show a visible "Skip" option or "Optional" badge. Users should not wonder if the Continue button is broken when they haven't filled anything
2. **Contextual examples (ONB-4):** Text input steps include placeholder examples or inspiration text (e.g., "Your Audience" shows: *"Example: Small business owners in Portland looking for affordable legal help"*)
3. **Layout variation by step type (ONB-5):** Text input steps are compact/centered; selection steps (Pages, Design, Tone) use wider layouts with visual cards; upload steps (Brand) show prominent drop zones

## Acceptance Criteria

- [ ] Optional steps have visible "Skip" or "Optional" label
- [ ] Text input steps include contextual example text
- [ ] Layout varies by step type (compact for text, wide for selection, prominent for upload)
- [ ] All existing onboarding tests pass
