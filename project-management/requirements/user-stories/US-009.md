# US-009: Onboarding Step 5 — Dynamic Industry Questions

**Epic:** Onboarding Wizard
**Priority:** P0
**Milestone:** M1 — Platform Foundation & Onboarding

## User Story
As a new user, I want to answer industry-specific questions generated by the AI, so that my website addresses the unique needs of my business type.

## Acceptance Criteria
- [ ] Given Step 5 of onboarding, when the page loads, then the AI Agent generates 3–5 questions based on the selected industry and business context
- [ ] Given the generated questions, when displayed, then each uses an appropriate input type (yes/no toggle, text field, multi-select as needed)
- [ ] Given a Healthcare industry selection, when questions generate, then they include relevant topics like appointments, locations, provider types
- [ ] Given a Legal industry selection, when questions generate, then they include practice areas, case types, consultation booking
- [ ] Given "Other" industry, when questions generate, then AI uses the free-text description to infer relevant questions
- [ ] Given all required questions are answered, when the user clicks "Generate My Website", then the onboarding data is finalized and generation begins
- [ ] Given the question generation takes time, when the AI is processing, then a loading state is shown (not a blank screen)

## Dependencies
- US-006 (Industry Selection)
- US-008 (Business Context)

## Technical Notes
- FR-105
- This is the first AI Agent interaction in the flow — uses Drupal AI Agents module
- Industry Agent determines questions based on industry taxonomy + business context
- Question schema must be structured (not free-form AI output) to render proper form elements
- Latency target: questions should appear within 3 seconds of page load
