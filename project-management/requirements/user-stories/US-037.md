# US-037: Custom Page Addition

**Epic:** Onboarding Enhancements
**Priority:** P1
**Milestone:** M6 — Onboarding Enrichment
**PRD Refs:** FR-107

## User Story
As a business owner, I want to add custom pages beyond the AI suggestions during onboarding, so that my site includes pages specific to my business needs.

## Acceptance Criteria
- [ ] Given the page selection step, when the user wants a page not in the AI suggestions, then they can click "Add Custom Page"
- [ ] Given the add custom page UI, when the user interacts with it, then they can enter a page title and a brief description of what should appear on that page
- [ ] Given custom pages, when more than 3 are attempted, then the system prevents adding more and shows a message explaining the limit
- [ ] Given custom pages, when onboarding completes, then custom pages are included alongside AI-suggested pages in the content pipeline with their user-provided descriptions

## Dependencies
- US-036 (Enhanced AI Page Suggestions)

## Technical Notes
- Custom page descriptions serve as the content generation prompt for that page
- Maximum 3 custom pages to keep generation time reasonable
- Custom pages should be visually distinguished from AI-suggested pages in the selection UI
