# US-094: Archie Brand Identity & Activity Log Polish

**Epic:** UX Polish
**Priority:** P1
**Milestone:** M20 — AI Transparency

## User Story
As a user going through onboarding, I want Archie's name to be visually distinct and the activity log to feel alive while steps are running, so that the AI assistant feels like a real character guiding me.

## Acceptance Criteria
- [ ] Every occurrence of "Archie" in the UI is rendered in brand-500 (indigo) color
- [ ] Archie's Workshop panel shows a typing/streaming indicator while a pipeline phase is `in_progress`
- [ ] The streaming indicator disappears when the phase completes or a new message arrives
- [ ] Indicator respects `prefers-reduced-motion` (static dots instead of animation)

## Dependencies
- US-093 (ActivityLog component must exist)

## Technical Notes
- Archie highlight: search all components for "Archie" text, wrap in `<span className="text-brand-400">Archie</span>`
- Streaming indicator: add a "typing dots" animation at the bottom of the ActivityLog when the latest phase is `in_progress` and no new message has arrived in >1s
