# US-008: Onboarding Step 4 — Business Context

**Epic:** Onboarding Wizard
**Priority:** P0
**Milestone:** M1 — Platform Foundation & Onboarding

## User Story
As a new user, I want to describe my services, target audience, competitors, and key calls-to-action, so that the AI generates relevant, targeted website content.

## Acceptance Criteria
- [ ] Given Step 4 of onboarding, when the user views the page, then they see fields for: services/offerings, target audience, competitors, key CTAs
- [ ] Given the services field, when a user enters text, then they can list multiple services (comma-separated or multi-line, up to 10)
- [ ] Given the target audience field, when the user types, then it accepts a free-text description (max 500 chars)
- [ ] Given the competitors field, when the user types, then they can enter competitor names or URLs (optional, up to 3)
- [ ] Given the CTAs field, when the user types, then they can enter desired call-to-action phrases (e.g., "Book Now", "Get a Quote")
- [ ] Given the user fills in at least the services field, when they click "Next", then data is saved and they proceed to Step 5

## Dependencies
- US-007 (Brand Input)

## Technical Notes
- FR-104

- CTA phrases used by the Content Agent for button text and placement
- Competitor URLs used as AI context (tone/positioning reference), not scraped
