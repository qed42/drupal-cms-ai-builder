# US-025: Media Management

**Epic:** Review & Editing
**Priority:** P2
**Milestone:** M3 — Site Editing & Refinement

## User Story
As a user editing my site, I want to upload and manage images for my website, so that I can replace placeholder images with my own photos and branding assets.

## Acceptance Criteria
- [ ] Given the Canvas editor, when the user clicks on an image component, then they see options to upload a new image or choose from the media library
- [ ] Given an image upload, when the user selects a file, then it accepts PNG, JPG, WebP up to 10MB
- [ ] Given uploaded images, when stored, then they are organized in the Drupal media library
- [ ] Given an uploaded image, when placed on a page, then it is automatically optimized (responsive sizes, WebP conversion)

## Dependencies
- US-020 (Canvas Page Editor)

## Technical Notes
- FR-306
- Drupal Media module for media management
- Image styles for automatic responsive image generation
- Consider integration with Unsplash API for stock images as a future enhancement
