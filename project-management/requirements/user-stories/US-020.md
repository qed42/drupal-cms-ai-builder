# US-020: Canvas Page Editor Integration

**Epic:** Review & Editing
**Priority:** P0
**Milestone:** M3 — Site Editing & Refinement

## User Story
As a user reviewing my generated site, I want to edit pages using the Drupal Canvas visual editor, so that I can customize layouts by dragging, dropping, and rearranging SDC components.

## Acceptance Criteria
- [ ] Given a generated site, when the user clicks "Edit" on any page, then the Drupal Canvas editor opens with the page loaded
- [ ] Given the Canvas editor is open, when the user views the page, then all SDC components are individually selectable
- [ ] Given a selected component, when the user drags it, then they can reorder components within the page
- [ ] Given the Canvas editor, when the user clicks "Add Section", then they see available Space SDC components to insert
- [ ] Given edits are made, when the user clicks "Save", then changes are persisted to the Drupal node

## Dependencies
- US-013 (Page Generation — pages must exist to edit)
- Drupal Canvas module installed and configured

## Technical Notes
- FR-301
- Canvas must be configured to use the Space theme's SDC component library
- User role/permissions must allow Canvas editing but restrict Drupal admin access
- Consider a simplified Canvas toolbar for non-technical users (hide developer-oriented options)
