# US-101: Designer Agent — AI-Generated Custom Sections

**Epic:** Code Component Generation
**Priority:** P0
**Milestone:** M26 — Code Component Generation

## User Story
As a user who selected "Unique & Creative" mode, I want the AI to generate custom-designed React/Tailwind sections for each page of my site, so that I get a one-of-a-kind website with bespoke layouts, animations, and visual treatments.

## Acceptance Criteria
- [ ] Given a section brief (heading, content, type, position, brand tokens, tone), when the Designer Agent runs, then it produces valid JSX + Tailwind CSS for that section
- [ ] Given the Designer Agent, when generating sections, then it supports at least 9 section types: hero, features, services, testimonials, about, team, gallery, CTA, and FAQ
- [ ] Given brand tokens (colors, fonts, logo URL), when a section is generated, then the output incorporates brand colors as Tailwind theme values and uses the specified fonts
- [ ] Given animation preferences ("subtle", "moderate", "dramatic"), when a section is generated, then animation complexity scales accordingly and all animations respect `prefers-reduced-motion`
- [ ] Given the Designer Agent output, when validated, then it passes all Code Component validation rules (structure, accessibility, security, motion safety)
- [ ] Given a validation failure, when the retry loop fires, then the Designer Agent receives the error feedback and regenerates (max 2 retries) before failing gracefully
- [ ] Given a section's `previousSectionSummary`, when generating the next section, then the Designer Agent maintains visual rhythm (avoids repeating the same layout pattern)
- [ ] Given the generated JSX, when it references images, then placeholder URLs follow the `/placeholder/{width}x{height}` convention for later image enhancement
