# US-005: Onboarding Step 1 — Site Basics Input

**Epic:** Onboarding Wizard
**Priority:** P0
**Milestone:** M1 — Platform Foundation & Onboarding

## User Story
As a new user, I want to enter my business name, tagline, logo, and contact email, so that the AI can personalize my website with my brand identity.

## Acceptance Criteria
- [ ] Given the onboarding wizard starts, when the user sees Step 1, then they see fields for: site name (required), tagline (optional), logo upload (optional), admin email (pre-filled from registration)
- [ ] Given the site name field, when the user types a name, then it is validated (min 2 chars, max 100 chars)
- [ ] Given the logo upload, when a user uploads an image, then it accepts PNG, JPG, SVG formats up to 5MB
- [ ] Given all required fields are filled, when the user clicks "Next", then data is saved and they proceed to Step 2
- [ ] Given the step is displayed, when the user views it, then they see a progress indicator showing "Step 1 of 5"

## Dependencies
- US-001 (User Registration)

## Technical Notes
- FR-101, FR-107
- Wizard built as a multi-step form (Drupal Form API or React-based frontend)
- Data stored as a "site profile" entity linked to the user account
- Logo stored via Drupal file system with appropriate image styles
