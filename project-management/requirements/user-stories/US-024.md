# US-024: Component Swap

**Epic:** Review & Editing
**Priority:** P2
**Milestone:** M3 — Site Editing & Refinement

## User Story
As a user editing my site, I want to swap one component for another compatible component from the design system, so that I can change the visual style of a section without losing its content.

## Acceptance Criteria
- [ ] Given a selected component in Canvas, when the user clicks "Swap Component", then they see a list of compatible alternative components
- [ ] Given compatible alternatives, when displayed, then only SDC components that accept similar data/props are shown
- [ ] Given a swap is selected, when applied, then the existing content is mapped to the new component's fields
- [ ] Given content cannot be fully mapped, when the swap happens, then unmapped fields are highlighted for the user to fill

## Dependencies
- US-020 (Canvas Page Editor)

## Technical Notes
- FR-304
- Requires a component compatibility matrix — which components can substitute for which
- Content mapping between components may need AI assistance for non-trivial mappings
