# US-033: Industry-Specific Follow-up Questions

**Epic:** Onboarding Enhancements
**Priority:** P0
**Milestone:** M6 — Onboarding Enrichment
**PRD Refs:** FR-101

## User Story
As a business owner, I want to answer industry-specific follow-up questions during onboarding, so that the AI can generate content that accurately reflects my business.

## Acceptance Criteria
- [ ] Given the system detects the user's industry from their idea input, when the user proceeds past the idea step, then 2-4 targeted follow-up questions are presented
- [ ] Given a healthcare business, when follow-up questions appear, then they include questions like "What are your top 3 specialties?" and "Do you accept insurance?"
- [ ] Given a legal business, when follow-up questions appear, then they include questions like "What are your practice areas?" and "Do you offer free consultations?"
- [ ] Given a restaurant business, when follow-up questions appear, then they include questions like "What cuisine type?" and "Do you offer delivery/catering?"
- [ ] Given follow-up questions, when they are configured, then they are defined in a JSON/config file per industry — not hardcoded in components
- [ ] Given the user answers follow-up questions, when they proceed, then answers are stored and passed to the content generation pipeline

## Dependencies
- US-006 (Industry Selection)

## Technical Notes
- Requires an industry-questions config file (JSON) mapping industry codes to question sets
- Current onboarding has 7 steps; this adds step 8 (follow-up questions)
- Questions should have input type metadata (text, select, multi-select) for flexible rendering
- Consider a fallback set of generic questions for unrecognized industries
