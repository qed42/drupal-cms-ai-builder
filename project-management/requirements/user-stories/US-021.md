# US-021: Section-Level AI Regeneration

**Epic:** Review & Editing
**Priority:** P0
**Milestone:** M3 — Site Editing & Refinement

## User Story
As a user editing my site, I want to select a section and ask the AI to regenerate it with optional guidance, so that I can improve specific parts without affecting the rest of the page.

## Acceptance Criteria
- [ ] Given the Canvas editor is open, when the user selects a section/component, then they see a "Regenerate with AI" option
- [ ] Given the regenerate option is selected, when clicked, then a prompt input appears where the user can provide guidance (e.g., "Make it more professional", "Add more detail about our services")
- [ ] Given guidance is provided (or skipped), when the user confirms, then the AI regenerates only that section's content and layout
- [ ] Given regeneration completes, when the new content appears, then the user can accept or revert to the previous version
- [ ] Given a regeneration, when it runs, then only the selected section is affected — surrounding content is untouched

## Dependencies
- US-020 (Canvas Page Editor)

## Technical Notes
- FR-302
- Requires Canvas integration point — custom action button on component selection toolbar
- AI regeneration uses the same Content Agent with additional user guidance as context
- Revision system or temporary storage for the "revert" capability
- Consider backing up the section state before regeneration (NFR-16)
