# US-030: One-Click Publish

**Epic:** Publishing & Deployment
**Priority:** P0
**Milestone:** M5 — Publishing & Subscription

## User Story
As a user who is satisfied with my site, I want to publish it with a single click, so that it goes live immediately without any technical steps.

## Acceptance Criteria
- [ ] Given the user is reviewing their site, when they click "Publish", then all draft pages and content are published simultaneously
- [ ] Given a publish action, when executed, then the site becomes accessible to anonymous visitors on the platform subdomain
- [ ] Given a successful publish, when complete, then the user receives a confirmation with the live site URL
- [ ] Given a published site, when the user makes edits later, then they can re-publish changes with the same single-click action
- [ ] Given the site is published, when viewed publicly, then navigation, pages, forms, and branding all render correctly

## Dependencies
- US-029 (Draft Mode)
- US-002 (Free Trial — must be active)

## Technical Notes
- FR-501
- Bulk publish action — iterate all draft nodes and set to "published"
- Platform subdomain assignment (e.g., {site-name}.platform.com)
- Clear Drupal cache after publish to ensure fresh content is served
