# US-023: Page Add & Remove

**Epic:** Review & Editing
**Priority:** P0
**Milestone:** M3 — Site Editing & Refinement

## User Story
As a user editing my site, I want to add new pages or remove generated pages, so that I can customize the site structure to match my needs.

## Acceptance Criteria
- [ ] Given the site editor, when the user clicks "Add Page", then they can create a new blank page with a title and choose a template/layout
- [ ] Given a new page is added, when created, then it appears in the site navigation automatically
- [ ] Given an existing page, when the user chooses "Delete Page", then a confirmation dialog appears
- [ ] Given a page deletion is confirmed, when executed, then the page is removed from the site and navigation
- [ ] Given the Home page, when the user attempts to delete it, then deletion is blocked with a message ("Home page cannot be deleted")

## Dependencies
- US-020 (Canvas Page Editor)

- FR-305
