# US-007: Onboarding Step 3 — Brand Input

**Epic:** Onboarding Wizard
**Priority:** P0
**Milestone:** M1 — Platform Foundation & Onboarding

## User Story
As a new user, I want to provide my brand colors, font preferences, and reference websites, so that the AI generates a site that matches my brand identity.

## Acceptance Criteria
- [ ] Given Step 3 of onboarding, when the user views the page, then they see: color palette picker, font selector, reference URL input, brand guidelines upload
- [ ] Given the color picker, when the user selects colors, then they can set primary, secondary, and accent colors (with sensible defaults)
- [ ] Given the font selector, when the user views options, then they can choose from a curated list of web-safe/Google Fonts pairings (heading + body)
- [ ] Given the reference URL field, when a user enters a URL, then it is stored for the AI to analyze (up to 3 URLs)
- [ ] Given the brand guidelines upload, when a user uploads a file, then it accepts PDF, PNG, JPG up to 10MB
- [ ] Given all inputs are optional, when the user skips all fields and clicks "Next", then sensible AI-chosen defaults are applied

## Dependencies
- US-006 (Industry Selection)

## Technical Notes
- FR-103
- Design tokens (CSS custom properties) generated from color/font selections
- Reference URLs passed to the AI agent for style analysis — not scraped, used as context in prompts
- Defaults should be industry-aware (e.g., blue/trust tones for healthcare, green for real estate)
