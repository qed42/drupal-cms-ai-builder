# US-031: Custom Domain Support

**Epic:** Publishing & Deployment
**Priority:** P2
**Milestone:** M5 — Publishing & Subscription

## User Story
As a business owner with a published site, I want to connect my own domain name, so that my website appears professional under my brand's URL.

## Acceptance Criteria
- [ ] Given a published site, when the user navigates to domain settings, then they see instructions for connecting a custom domain
- [ ] Given domain setup instructions, when displayed, then they include DNS configuration steps (CNAME or A record)
- [ ] Given correct DNS configuration, when the domain is verified, then the site becomes accessible on the custom domain
- [ ] Given a custom domain is connected, when accessed, then SSL is automatically provisioned (US-032)
- [ ] Given a custom domain, when the platform subdomain is also accessed, then it redirects to the custom domain

## Dependencies
- US-030 (One-Click Publish)

## Technical Notes
- FR-503
- DNS verification via TXT record or CNAME check
- Requires server/hosting infrastructure configuration — this may be partially deferred depending on OQ-5 (hosting infra)
