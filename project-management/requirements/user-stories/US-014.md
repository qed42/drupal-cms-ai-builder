# US-014: SEO Content Generation

**Epic:** AI Website Generation
**Priority:** P0
**Milestone:** M2 — AI Site Generation Engine

## User Story
As a business owner, I want the AI to generate SEO-optimized content for my pages, so that my website ranks well in search engines from day one.

## Acceptance Criteria
- [ ] Given a generated page, when the content is created, then it includes proper H1, H2, H3 heading hierarchy
- [ ] Given a generated page, when viewed in Drupal, then it has a meta title (< 60 chars) and meta description (< 160 chars)
- [ ] Given generated content, when analyzed, then it uses industry-relevant keywords naturally (not keyword-stuffed)
- [ ] Given generated pages, when inspected, then image elements have alt text placeholders describing expected content
- [ ] Given the business context, when content is generated, then the tone matches the industry (professional for legal, warm for healthcare, energetic for restaurants)

## Dependencies
- US-013 (Page Generation)
