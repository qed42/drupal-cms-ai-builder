# US-002: Free Trial Activation

**Epic:** User Account & Trial Management
**Priority:** P0
**Milestone:** M1 — Platform Foundation & Onboarding

## User Story
As a newly registered user, I want to automatically receive a 14-day free trial, so that I can build and preview my website before committing to a subscription.

## Acceptance Criteria
- [ ] Given a new user registers, when their account is created, then a 14-day trial period begins automatically
- [ ] Given an active trial, when the user logs in, then they see remaining trial days in the UI
- [ ] Given an active trial, when the user uses any feature, then all features are fully accessible (no feature gating)
- [ ] Given a trial, when it is created, then the trial expiry date is stored on the user account

## Dependencies
- US-001 (User Registration)

## Technical Notes
- Custom field on user entity to store trial start/end dates
- Trial status service that checks expiry across the platform
- No payment information required to start trial
