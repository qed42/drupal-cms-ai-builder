# US-019: Generation Progress UI

**Epic:** AI Website Generation
**Priority:** P1
**Milestone:** M2 — AI Site Generation Engine

## User Story
As a user who just clicked "Generate My Website", I want to see real-time progress of what the AI is doing, so that I know the system is working and approximately how long to wait.

## Acceptance Criteria
- [ ] Given the user clicks "Generate My Website", when generation starts, then a progress screen appears with step indicators
- [ ] Given generation is in progress, when each major step completes, then the progress updates (e.g., "Analyzing your industry...", "Creating content structure...", "Building pages...", "Applying your brand...")
- [ ] Given generation is in progress, when the user sees the progress screen, then they cannot navigate away without a confirmation dialog
- [ ] Given generation completes successfully, when all steps finish, then the user is automatically redirected to a preview of their generated site
- [ ] Given generation fails at any step, when an error occurs, then the user sees a friendly error message with a "Retry" button

## Dependencies
- US-012 through US-018 (All generation stories)

## Technical Notes
- FR-208
- Server-Sent Events (SSE) or WebSocket for real-time progress streaming
- Alternative: polling endpoint that returns current generation step
- Generation steps run as a Drupal Queue/Batch API process with status tracking
