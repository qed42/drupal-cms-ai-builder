# US-006: Onboarding Step 2 — Industry Selection

**Epic:** Onboarding Wizard
**Priority:** P0
**Milestone:** M1 — Platform Foundation & Onboarding

## User Story
As a new user, I want to select my industry from predefined categories, so that the AI can generate industry-relevant content, pages, and compliance considerations.

## Acceptance Criteria
- [ ] Given Step 2 of onboarding, when the user views the page, then they see industry options: Healthcare, Legal, Real Estate, Restaurant, Professional Services, Other
- [ ] Given the industry options, when the user selects one, then it is visually highlighted as selected
- [ ] Given the "Other" option, when selected, then a free-text field appears for the user to describe their industry/use-case
- [ ] Given an industry is selected, when the user clicks "Next", then the selection is saved and they proceed to Step 3
- [ ] Given no industry is selected, when the user clicks "Next", then a validation message appears

## Dependencies
- US-005 (Site Basics Input)

## Technical Notes
- FR-102
- Industry taxonomy vocabulary in Drupal with predefined terms
- Industry selection drives the AI Agent's dynamic question generation in Step 5 (US-009)
- "Other" free-text stored as a custom field on the site profile entity
