# US-022: Content Inline Editing

**Epic:** Review & Editing
**Priority:** P0
**Milestone:** M3 — Site Editing & Refinement

## User Story
As a user editing my site, I want to click on any text and edit it directly on the page, so that I can make quick content changes without navigating to a separate edit form.

## Acceptance Criteria
- [ ] Given the Canvas editor is open, when the user clicks on a text element, then it becomes editable in place
- [ ] Given inline editing is active, when the user types, then the text updates in real-time on the page
- [ ] Given inline editing, when available, then it supports basic formatting (bold, italic, links)
- [ ] Given inline edits are made, when the user clicks "Save", then all text changes are persisted

## Dependencies
- US-020 (Canvas Page Editor)

## Technical Notes
- FR-303

- CKEditor integration for rich text inline editing
